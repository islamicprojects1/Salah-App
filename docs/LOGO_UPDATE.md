# لماذا لا يتغيّر لوقو التطبيق؟

## مسارات اللوقو

| الاستخدام | المسار |
|-----------|--------|
| داخل التطبيق (Splash, Login, Onboarding) | `assets/icons/salah_app_logo.png` |
| أيقونة التطبيق (Android/iOS على الشاشة الرئيسية) | نفس الملف عبر `flutter_launcher_icons` |

## سبب عدم التحديث

### 1. اللوقو داخل التطبيق
- تأكد أن الملف الجديد في: `assets/icons/salah_app_logo.png`
- **Hot Reload لا يحدّث الـ assets** — تحتاج **Hot Restart** أو إعادة تشغيل التطبيق كاملاً

### 2. أيقونة التطبيق (على الشاشة الرئيسية)
- تغيير الملف وحده **لا يكفي**
- يجب تشغيل أمر توليد الأيقونات:

```bash
dart run flutter_launcher_icons
```

ثم إعادة تثبيت التطبيق على الجهاز/المحاكي.

### 3. التخزين المؤقت
- أحياناً الجهاز أو المحاكي يخزّن الأيقونة القديمة
- جرّب: حذف التطبيق وإعادة تثبيته، أو مسح بيانات المحاكي
